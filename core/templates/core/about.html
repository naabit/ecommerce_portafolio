Acerca de este proyecto

Purple Store es una aplicación e-commerce desarrollada como proyecto final de formación en desarrollo Full Stack con Python.

El sistema permite a los usuarios explorar un catálogo de productos, gestionar un carrito de compras y completar un proceso de checkout, todo con persistencia en base de datos.

El proyecto incluye autenticación de usuarios, control de acceso por roles (cliente, staff y administrador) y una interfaz construida con Bootstrap, enfocada en ofrecer una experiencia clara y funcional.

Tecnologías utilizadas

  • Django (framework backend)
  • ORM de Django y base de datos relacional
  • Bootstrap 5 (frontend)
  • HTML, CSS y JavaScript
