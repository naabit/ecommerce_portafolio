Purple Store

Un e-commerce desarrollado con Django que integra catálogo dinámico, autenticación de usuarios, carrito de compras y flujo de checkout con persistencia en base de datos.

Catálogo dinámico

Los productos se cargan desde la base de datos y se presentan en una interfaz clara, escalable y fácil de recorrer.

Carrito y checkout

El proyecto incluye un flujo completo de compra: agregar productos, revisar subtotales, confirmar pedido y registrar la orden.

Usuarios y roles

Se implementa autenticación con acceso diferenciado para clientes, personal staff y administración del sistema.

¿Qué demuestra este proyecto?

Purple Store fue desarrollado como proyecto de portafolio para integrar distintas capas de una aplicación web real: backend con Django, persistencia con ORM, autenticación, vistas, templates y una interfaz responsiva construida con Bootstrap.

El foco del proyecto está en ofrecer una experiencia funcional, clara y mantenible, siguiendo una estructura modular por aplicaciones.
